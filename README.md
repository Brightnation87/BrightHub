# BrightHub

BrightHub is an all-in-one learning and coding platform prototype designed to help learners study, practice, and experiment in one unified environment. It brings together learning content, a code-focused workspace, and the foundation for AI-assisted guidance in a mobile-friendly interface.

This project was built and submitted for the **Missing Amps Winter Classic Hackathon**.
